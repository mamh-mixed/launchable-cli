= Object model
:slug: object-model

{PRODUCT}'s object model includes concepts that reflect objects in your development/CI environment (like xref:concepts:build.adoc[Build] and xref:concepts:test-session.adoc[Test Session]) alongside objects which are value-add concepts unique to {PRODUCT} like xref:concepts:subset.adoc[Subset] and xref:concepts:insight.adoc[Insight].

The diagram below illustrates how these objects relate to one another: which entities own others, which ones are associated only as context, and how information flows through the system. It is intended to help build a clear mental model of the product and understand the role of each entity you interact with.
