== Test Status

{PRODUCT} defines the following statuses. For example, let's say 100 tests were run, 10 failed, and 3 of them flaked. This means 90 tests passed, but 3 of them passed after a number of retries, and 10 tests kept failing throughout.

* Success: A test that passed.
* Failure: A test that failed in all executions.
* Flake: A test failure that eventually succeeded after a number of retries within the same test session.
* Skip: A test that was skipped.
